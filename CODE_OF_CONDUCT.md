# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text is available at the link above. In short:

- Be respectful and constructive in all interactions (issues, PRs, discussions, commit messages).
- Assume good intent, especially across language and timezone differences.
- Focus feedback on the technical content, not the person.
- No spam, no off-topic content, no personal attacks.

## Reporting

If you see behaviour that violates these guidelines, please report it privately:

- Open a confidential report via [GitHub Security Advisories](https://github.com/mobel8/vibe-term/security/advisories/new) for any sensitive matter.
- Otherwise, contact the maintainers listed in [`.github/CODEOWNERS`](./.github/CODEOWNERS).

Reports are handled confidentially. Maintainers will acknowledge receipt within 72 hours and follow up with next steps.

## Enforcement

Maintainers may take any action they consider appropriate, including warnings, temporary bans from interactions, or permanent removal from the project, in response to behaviour that does not align with this Code of Conduct.

## Scope

This Code of Conduct applies to all project spaces: the GitHub repository, its issues, pull requests, discussions, releases, and any associated chat channels or events organised by the project.
